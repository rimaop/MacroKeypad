# MacroKeypad

I created a macro keypad.

Using the Arduino Micro.

The keypad shortcuts that can be set in advance.

- 3x3 keypad
- 3LED

######Materials
- arduino Micro
- cherry mx switch

######included
- Arduino source
- Eagle CAD
